E.qianka.com

Well... Personally, this shouldn't be a good project, to be honest, because it was just a project for big money. I had accepted to charge the design of this project mainly because it was my first time to design a data base client -- I wanted to know if I could do good job with only numbers and functional description. And, thanks to my teammates, this project has left a happy memory.


- part 1 -

Main Pages

The home page consists of some boxes for data-monitoring, just like any other of this kind. I made great effort on the design of the margin between elements, to make it looks nice. Any way, I always remember an old saying by Pythagoras: Beauty is the proportion of harmony!


- part 2 -

Log in / Sign up

Thank to this project, I think I have discovered the fun of designing login/signup pages. They can be designed as simple as possible, and as clean as possible.


- part 3 -

Font
