GeoGebra Books

In this file, you will see an imaginary project designed based on GeoGebra, an online math tool. The introduction of the file might a little bit long, because it is an UI/UX design file and also a new function sheet. Hope enjoy!

Why did this?

This project is imaginary, totally, yet. It is a rebound design for GeoGebra.com, whose one of the main tool GeoGebra Classic 6 is the most important tool during my last year of math learning. I like it so much, because it allows me to visualize the functions and definitions in coordinates so quickly, which means it allows you to feel mathematics intuitively, which is an extremely crucial step to understand mathematics.

By any means, GeoGebra Classic 6 and some other GeoGebra tools is much more powerful than most of online tools for hobbies, and it can deal with much more complicated problems than Desmos can. But, maybe because of the complexity, GeoGebra also has a lot of problems I've found since the first time I use it. Firstly, it is unstable on outputs. Secondly, its UX is a nightmare, compared to some other tools like Sketch whom I am using every. Thirdly, the UX of math books and activities is also not as good as it should be, on both ends of reading and editing.
