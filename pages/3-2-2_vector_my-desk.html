iMac

This is an 27 inches iMac I brought in the year 2015. The desktop background is an oil-painting by Vincent van Gogh, who is one of my favorite artist. You can see there is a dark-grey window containing some codes. Well, it is Atom, the best editor I like until now. I use it as a Markdown + LaTeX editor to write my math notes. I am also using it to code my GitHub page, and a blog of math.

Well, I love editor, and Atom the bestÔºÅ









Magic board

The `a`, `z`, `i`, `space` and a few other key of this magic board are going wrong these days, because of a game called Dirt Rally. I set `a` and `z` for gear up and gear down, `i` and `space` as accelerator and brake, `tab` as clutch and `shift` as hand brake. That is why I have broken several magic boards these years.

But, as the old saying "no pay no gain", I won top 10 in a globe weekly event for Group A weeks ago.
